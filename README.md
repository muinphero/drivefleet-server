# DriveFleet — Car Rental Platform

Live Site:
https://drivefleet.muin.site

## Features

- Secure authentication with Email/Password and Google Login
- Explore available cars with search and filter
- Add, update, and delete car listings
- Book cars and manage bookings
- Protected private routes with HTTPOnly cookies
- Responsive design for mobile, tablet, and desktop

## Technologies

Frontend: Next.js, React, HeroUI
Backend: Express.js, MongoDB Atlas, Better Auth
Deployment: PM2, Nginx, Cloudflare
